# Padel Factory Global

High-end interactive website for Padel Factory UAE featuring a 3D exploded view of a padel court, material grid, and cost calculator.

## Project Structure

The code is organized into the following directories as requested:

- `/src/components/3DAssembly`: Contains the 3D models and scene logic using React Three Fiber.
- `/src/components/MaterialGrid`: Contains the industrial-grade materials showcase.
- `/src/components/IndustrialUI`: Contains the high-end UI components including the Magnetic Cursor, Cost Calculator, Header, and Footer.

## GitHub Integration

To push this code to your GitHub repository `padel-factory-global`, use the "Export to GitHub" feature in the AI Studio settings menu, or download the ZIP and push it manually.

## Technologies Used

- React 19
- Tailwind CSS v4
- Three.js & React Three Fiber
- GSAP & ScrollTrigger
- Lenis (Smooth Scrolling)
- Lucide React (Icons)
